功能特性

  • 智能OCR识别 - 支持各类银行单据的自动识别和数据提取，准确率达99.5%
  • 自动化审批流程 - 根据预设规则自动处理业务审批，支持多级审批

快速开始

1

安装MCP客户端

首先确保您的系统已安装MCP客户端和必要的依赖项

npm install -g @modelcontextprotocol/cli
pip install mcp-client python-dotenv
2

下载并配置

下载RPA文档处理器并配置环境变量

git clone https://github.com/rpastudio/rpa-document-processor.git
cd rpa-document-processor
cp .env.example .env
3

配置环境变量

编辑.env文件，配置必要的环境变量

# OCR服务配置
OCR_API_KEY=your_ocr_api_key
OCR_ENDPOINT=https://ocr.api.example.com

# 数据库配置
DB_HOST=localhost
DB_PORT=5432
DB_NAME=rpa_documents
DB_USER=rpa_user
DB_PASSWORD=your_password

# 审批流程配置
APPROVAL_WEBHOOK=https://your-approval-system.com/webhook
MAX_RETRY_ATTEMPTS=3
4

启动服务

启动RPA文档处理器服务

npm install
npm run build
npm start
5

连接测试

验证MCP连接是否正常

curl -X POST http://localhost:3000/health
# 返回: {"status": "healthy", "version": "1.5.2"}

API 接口文档

文档处理接口

POST /api/v1/documents/process

处理上传的文档，执行OCR识别和数据提取

请求参数
{
"file": "base64_encoded_file",
"document_type": "account_opening", // 文档类型
"options": {
"auto_approve": false, // 是否自动审批
"confidence_threshold": 0.95 // 置信度阈值
}
}
响应示例
{
"success": true,
"task_id": "task_12345",
"extracted_data": {
"customer_name": "张三",
"id_number": "110101199001011234",
"phone": "13800138000",
"account_type": "储蓄账户"
},
"confidence": 0.98,
"status": "pending_approval"
}

使用示例

Python 客户端示例

import requests
import base64

# 文档处理示例
def process_document(file_path, doc_type):
with open(file_path, 'rb') as file:
file_content = base64.b64encode(file.read()).decode('utf-8')

payload = {
'file': file_content,
'document_type': doc_type,
'options': {
'auto_approve': False,
'confidence_threshold': 0.95
}
}

response = requests.post(
'http://localhost:3000/api/v1/documents/process',
json=payload
)

return response.json()

# 使用示例
result = process_document('account_form.pdf', 'account_opening')
print(f"任务ID: {result['task_id']}")
print(f"提取数据: {result['extracted_data']}")

JavaScript 客户端示例

// 使用 fetch API 处理文档
async function processDocument(fileInput, documentType) {
const file = fileInput.files[0];
const reader = new FileReader();

reader.onload = async function(e) {
const base64File = e.target.result.split(',')[1];

const payload = {
file: base64File,
document_type: documentType,
options: {
auto_approve: false,
confidence_threshold: 0.95
}
};

try {
const response = await fetch('/api/v1/documents/process', {
method: 'POST',
headers: {
'Content-Type': 'application/json'
},
body: JSON.stringify(payload)
});

const result = await response.json();
console.log('处理结果:', result);

// 监控任务状态
pollTaskStatus(result.task_id);
} catch (error) {
console.error('处理失败:', error);
}
};

reader.readAsDataURL(file);
}

// 轮询任务状态
async function pollTaskStatus(taskId) {
const interval = setInterval(async () => {
const response = await fetch(`/api/v1/tasks/${taskId}`);
const task = await response.json();

console.log(`任务状态: ${task.status} (${task.progress}%)`);

if (task.status === 'completed' || task.status === 'failed') {
clearInterval(interval);
console.log('任务完成:', task.result);
}
}, 2000);
}
