使用文档

组件描述

TransferConfirmCard 是一个用于银行转账确认的LUI交互卡片组件。 该组件提供完整的转账信息展示，包括金额、收款人信息、付款账户信息等， 并提供确认、取消、修改等交互操作。

使用场景

个人转账流程
在个人转账的最终确认环节使用，展示完整的转账信息供用户确认
企业转账流程
在企业对公转账中使用，支持更多的审批和风险控制信息展示

代码示例

// 在复合意图中使用
const transferIntent = {
  component: 'TransferConfirmCard',
  props: {
    amount: 1000.00,
    payeeName: '李四海',
    payeeAccount: '6222219020095522901'
  },
  events: {
    onConfirm: handleConfirm,
    onCancel: handleCancel
  }
};

注意事项

金额参数必须为数字类型，组件内部会自动格式化显示
账号信息会自动进行脱敏处理，确保安全性
支持多种安全验证方式，可通过配置选择
