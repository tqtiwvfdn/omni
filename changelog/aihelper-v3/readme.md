## 格式说明

本更新日志遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/) 规范，并采用 [语义化版本](https://semver.org/lang/zh-CN/) 进行版本管理。

### 更新类型
- **Added** - 新增功能
- **Changed** - 功能变更
- **Fixed** - 问题修复
- **Removed** - 功能移除
- **Deprecated** - 已弃用功能
- **Security** - 安全相关更新
